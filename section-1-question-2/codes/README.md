# Code README

Run the analysis by opening `q2_coe_price_elasticity.ipynb` in Jupyter and running all cells.

The notebook automatically looks for `COEBiddingResultsPrices.csv` in `data/` (same folder as the notebook), the current folder, parent folders, or `/mnt/data`.

Main dependency list:

```bash
pip install pandas numpy matplotlib scikit-learn jupyter
```

Outputs are saved to the `../outputs/` folder (one level above `codes/`).
